# KVANT GET 🟢

**Kvant Get** — это легкая и эффективная утилита для загрузки видеоконтента с YouTube. Модуль является частью экосистемы физической нейросети **KVANT** и используется для формирования локальных обучающих выборок.

---

## 🚀 Функционал
* **Мгновенный захват:** Копируйте ссылку и нажимайте `Paste` — программа сама подтянет данные.
* **Визуальное подтверждение:** Система отображает превью видео перед началом загрузки.
* **Гибкая настройка путей:** Удобный выбор папки сохранения через графический интерфейс.
* **Кроссплатформенность:** Поддержка Windows и Linux.

---

## 🛠 Инструкция по использованию
1. Скопируйте ссылку на видео в браузере.
2. В интерфейсе программы нажмите кнопку **Paste**.
3. После появления превью нажмите **Download**.
*Текущий путь сохранения отображается в левом нижнем углу интерфейса.*

###💡 Как проверить сайт?

Если вы хотите узнать, поддерживается ли конкретный сайт, есть два способа:

1. Просто вставьте ссылку в вашу программу. Если yt-dlp узнает сайт, он тут же начнет подгружать название и превью.
2. Посмотреть официальный список. Он огромный, его можно найти по этой ссылке:[ Supported Sites](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md).

###⚠️ Важное уточнение

Для того чтобы видео с VK, Rutube, TikTok и других сайтов скачивались в максимально доступном качестве (и склеивались в один файл), крайне рекомендуется установить FFmpeg. Без него на некоторых сайтах программа может скачать только видео без звука или наоборот, так как современные сайты отдают их отдельными потоками.
